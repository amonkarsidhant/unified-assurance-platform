# Roadmap

## v1 (this starter)
- Policy scaffolding (gates + risk)
- Runnable local assurance scripts
- CI templates for API/web/event/security
- Evidence collection + markdown release report
- Sample service and result files

## v1.1
- JSON schema validation for policies/catalog
- Trend reporting across builds
- PR comment bot for gate decisions
- Standardized SARIF + JUnit ingestion

## v1.2
- Multi-repo rollup dashboard
- Environment-aware gates (dev/stage/prod)
- Risk-adaptive dynamic test selection
- Signed evidence attestations

## v2
- Agentic orchestrator for intelligent test prioritization
- Auto-remediation suggestions tied to failed gates
- Governance API for enterprise policy management
