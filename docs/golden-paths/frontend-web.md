# Golden Path: Frontend Web

## Baseline controls
- Lint + type checks
- Component and e2e smoke tests
- Accessibility checks (critical flows)
- Bundle size budget check

## Suggested commands
```bash
npm ci
npm run lint
npm run typecheck
npm run test
npm run test:e2e:smoke
```

## Required evidence
- Test pass/fail summary
- Accessibility violations report
- Build + bundle metrics
