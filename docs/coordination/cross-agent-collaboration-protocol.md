# Cross-Agent Collaboration Protocol (No-Silo Rule)

## Core principle

No sprint-critical workstream should rely on a single isolated agent.
Context, decisions, and evidence must be visible and transferable.

## Rules

1. **Shared context first** — intent/scope/assumptions in issue or PR artifacts.
2. **Primary + backup ownership** — every task has owner and reviewer/backup.
3. **Evidence over narrative** — claims link to inspectable artifacts.
4. **Handoff contract** — include state, blockers, risks, and next steps.
5. **Visible decisions** — GO/CONDITIONAL/NO-GO with rationale + approver.
6. **Exception hygiene** — owner, expiry, follow-up issue required.

## Disallowed anti-patterns

- Single-agent completion claims without evidence.
- Hidden side channels for release-critical decisions.
- Exception approvals without expiry/owner.
