# API Sample Service Descriptor

This folder represents a backend service onboarding to UAP.

Expected scripts in service repo:
- `npm run lint`
- `npm test`
- `npm run test:integration`
- `npm run test:contract`
- `npm run test:security`
- `npm run test:perf:smoke`

For this starter repo, assurance scripts use simulated command outputs if these scripts are unavailable.
