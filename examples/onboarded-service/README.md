# Example: Onboarded Service Bundle

This folder is a complete sample output pack for one service onboarding.

- `service-profile.json`  
  Example service configuration created by `make onboard`.

- `generated-onboarding.md`  
  Example onboarding checklist document generated under `docs/generated/`.

- `preflight-summary.json`  
  Example preflight result showing checks, escalation reason, and pass/fail.

- `failure-explanation-and-next-steps.md`  
  Example operator-facing explanation for a failed gate and practical next actions.

- `promotion-decision.json`  
  Example promotion gate decision payload (GO/NO-GO context).

Use these as reference templates when onboarding your own service.
