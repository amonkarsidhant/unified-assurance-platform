# Minimum Dashboard Set

## 1) Release Health
- Gate pass/fail trend by repo
- GO/CONDITIONAL/NO-GO counts
- Top failed controls (last 14 days)

## 2) Test Effectiveness
- Pass rate by test category
- Flaky test count and churn
- Escaped defects vs pre-release failures

## 3) Reliability + Performance
- Availability SLO attainment
- p95 latency by critical endpoint
- Error budget burn for latest release

## 4) Security Posture
- High/critical vulnerabilities over time
- Mean time to remediate by severity
- Secret scan violations
